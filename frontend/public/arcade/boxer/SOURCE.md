# Boxer Game Character

The arcade fighter frames are adapted from **Boxer Game Character** by Raga2D.

- Source: <https://opengameart.org/content/boxer-game-character>
- Original download: <https://opengameart.org/sites/default/files/boxer_game_sprite_oga.zip>
- License: [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/)

The original pack supplies idle, walk, punch, uppercut, block, hurt, and KO
sequences. Agent Rumble selects representative frames, downsamples them to
186×178 PNG files, and renames the uppercut sequence to `special` for the game
runtime. No project data or branding is embedded in these images.
