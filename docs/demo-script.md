# Agent Rumble Script

Agent Rumble is an evidence-backed discovery and comparison for agent-related
software projects. It helps people and agents search a prepared catalog, build
a shortlist, compare projects for their use case.

## What We Build

We use Codex with the Agent Project Card skill to analyze agent-related software
projects and create standardized, evidence-backed Agent Project Cards.

## Why We’re Building This

Coding agents are reshaping how software gets built—but as the ecosystem grows,
it is becoming harder to understand what each agent does and how they differ.

Inspired by model cards and data cards, the Agent Project Card creates a shared
standard for describing agents—making them easier to discover, compare,
evaluate, and benchmark.
