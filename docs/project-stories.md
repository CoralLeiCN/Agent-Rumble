# Our Build Stories

These are our notes about how we build Agent Rumble and Agent Project
Intelligence. They are not a source of truth.

## Codex as the Core Harness

We use Codex heavily. Codex is the core harness for our tools.

## Tools as Skills

We provide our tools as skills. A Codex plugin is the first supported form.

## Tools as Services

We also provide our tools as services. The services leverage Codex as the core
harness.
