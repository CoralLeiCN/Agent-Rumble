# Writing Guidelines

## User Requirements

Rephrase user requirements into a clearer, more structured form before recording
them under the relevant topic in `docs/requirements.md`.

The rephrased version must preserve what the user specified and must not add new requirements, constraints, or details.

## Chat

Use the same clear, structured writing approach in chat.

Do not add writing rules that the user has not specifically requested.

Do not add content beyond what the user asked to write.
